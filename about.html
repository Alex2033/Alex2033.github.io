Александр Хохлов

HTML-верстальщик, Web-разработчик

Здравствуйте, я занимаюсь HTML-версткой сайтов: интернет-магазинов, Landing Page, флаеров. Увлекаюсь версткой около года и стремлюсь к новым знаниям. Соотвественно владею HTML5/CSS3. Имею опыт адаптивной кроссбраузерной (IE8 +, Chrome, Firefox, Safari, Opera, Edge) верстки. Верстаю, соблюдая валидность кода по стандартам W3C. Есть опыт написания анимаций на CSS3. Для добавления интерактивности сайту использую jQuery. Знаю базовый синтаксис javascript. В дальнейшем планирую основательно изучать JS и его фреймворки.

В своей работе я использую сборщик проектов Gulp для автоматизации рутинных, часто повторяющихся задач. Для ускорения написания CSS-кода использую препроцессоры такие, как Stylus, Sass/SCSS, Less. В основном пользуюсь Sass. Также для HTML препроцессор Pug.

Делаю адаптивную верстку с помощью сетки Bootstrap 3,4. Помимо этого умею делать адаптив с помощью Flexbox, CSS-grid. Верстаю сайты блоками. Для быстрой и легкой разбработки интерфейсов наименовываю блоки на основе БЭМ(Блок, Элемент, Модификатор).
